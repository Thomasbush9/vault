# Lab

Each subfolder is one project = one "lab member". Each project has both:
- this folder (notes, logs, lit, decisions, data pointers — synced via the vault)
- a separate GitHub repo where Claude Code is launched (linked from `repo.md` in each folder)

## Active

- `protein-interp/` — mechanistic interpretability of cofolding diffusion models (Boltz-2, AF3, ESMFold).
- `protforge/` — pipeline / tooling used by `protein-interp`.

## Idea-stage

- `environmental-perturbation/` — how solvent / temperature perturbations reshape protein energy landscapes.

## Adding a new project

Copy `Rules/project-template/` into `Lab/<name>/`, then fill in `CLAUDE.md`, `README.md`, `agenda.md`, `repo.md`.
