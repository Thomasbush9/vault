
## Neural Machine Translation: 

Earlier paradigm for machine translation: encoder-decoder. The encoder encodes a sentence into a fixed-length vector from which a decoder generates a translation. 

Main issue: the fixed vector is a bottleneck. The built a model that automatically searches for parts of a source sentence that are relevant for the given word.
