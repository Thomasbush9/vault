# Knowledge

Atomic, evergreen concept notes. One topic per file, ~1 screen each, cross-linked via Obsidian `[[wikilinks]]`. Not project-bound — these are the building blocks projects pull from.

## Topics

- `BIO/` — biology, protein structure, single-cell, related metrics (e.g. TM-score).
- `CS/` — ML / DL concepts, kernels, interpretability primitives (e.g. model diffing), generative models (diffusion, RNN-AE, contrastive).
- `Math/` — calculus, linalg, probability, info theory, statistics, signal processing, function-property atoms (tanh, monotonic, etc.).
- `NLP/` — NLP-specific notes and course materials.
