# Notes vault

Canonical Obsidian vault, synced via git/GitHub across devices.

## Layout

- `Lab/` — projects ("lab members"). Each = an Obsidian folder + a separate GitHub repo (cross-linked).
- `Knowledge/` — atomic concept notes by topic (BIO, CS, NLP, Math).
- `Rules/` — vault spec (`Knowledge base config.md`) + project template + Claude conventions.
- `inbox/` — daily notes, raw drafts, things to file later.
- `archive/` — parked projects and old atoms (created on demand).

## Where to start

- `Rules/Knowledge base config.md` — full spec for the cross-device setup.
- `Lab/README.md` — index of active projects.
- `Knowledge/README.md` — index of topic atoms.
